## Projektöversikt

**Problem:** Användare har svårt att planera och estimera AI-sprint-projekt.

**Målgrupp:** Solo-utvecklare och små team som bygger med AI-verktyg.

**Typ:** saas-dashboard

---

## Scope (MVP)

1. WHY/WHAT/HOW-flöde för projektdefinition
2. Fasplan-generator med credit-estimat
3. PDR-export till Markdown och GitHub

---

## Faser att bygga


### Fas 1: Foundation & Setup

Grundläggande projektstruktur och Supabase-koppling.

**Tasks:**
- Setup Vite + React + Tailwind (new_component)
- Konfigurera Supabase-klient (integration)


### Fas 2: Sprint Flow MVP

Chat-baserat WHY/WHAT/HOW-flöde med fasplan-generering.

**Tasks:**
- useSprintFlow hook (logic)
- ChatPage UI (new_flow)


### Fas 3: Export & Handoff

PDR-export, GitHub-integration och handoff-filer.

**Tasks:**
- PDRExportDialog (new_component)
- GitHub Export Service (integration)
